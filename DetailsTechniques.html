Détails Technique

Pour la réalisation de ce site, nous avons choisi d'utiliser 10 pages HTML et 1 page de CSS; différentes pages aurait pu être supprimé avec un code plus dynamique (Java CSS) comme par exemple la page du panier plein qui pourrait être juste un message d'erreur lorsque le panier est vide et non une page entière.

Sur la fiche technique des produits, nous aurions aimer pouvoir faire un zoom sur les articles mais cette fonctionnalité se réalise également avec le langage Java.

Nous avons intégré des polices personnalisées que nous avons téléchargées sur internet et nous avons utilisé différentes images ainsi qu'une vidéo dans notre code
