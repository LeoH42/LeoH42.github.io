Drone de course

VELOCITY : VOTRE MACHINE DE COURSE

  • Vitesse : 110km/h
  • Poids : 300g
  • Autonomie : 15 minutes
  • Portée : 200m
  • Prix : 349.99€

Le Velocity est idéal pour une introduction au monde de la course de drones. Ses moteurs brushless offrent une puissance de propulsion optimale qui permet à votre drone d’atteindre les 12m/s. Vous aurez entre les mains un aéronef totalement équilibré entre vitesse et maniabilité.
Le Velocity est équipé d’une radio-commande de longue portée vous permettant de le contrôler jusqu’à 300 mètres. Celle-ci vous garantit une sécurité en vous alertant si le drone est hors de portée ou si sa batterie est faible.
Le Velocity est équipé d’une caméra intégrée de haute qualité qui vous permet d’enregistrer facilement toutes vos aventures. Il est recommandé d’utiliser des lunettes FPV (non fournies) afin d’optimiser votre expérience de vol pour une immersion totale.
